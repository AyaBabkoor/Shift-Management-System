# Shift Management System

A full-stack web application developed for King Abdulaziz Hospital to digitize staff scheduling, attendance tracking, and shift management through automated workflows and role-based access.

> ⚠️ Source code is private due to organizational confidentiality.

---

## Technologies

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MySQL
- Git & GitHub

---

## My Contribution

As a Full-Stack Development Intern, I contributed to:

- Front-end and back-end development
- Database design
- REST API implementation
- Feature development and testing
- Deployment to the hospital's internal production server

---

## Key Features

- Shift scheduling and roster management
- Attendance tracking
- Shift swap and edit requests
- Approval workflows
- Role-Based Access Control (RBAC)
- Audit logs
- In-app notifications
- Bilingual interface (Arabic/English)
- Operational reports
- Responsive design
